# OptiFeat

OptiFeat یک پلتفرم ماژولار برای انتخاب ویژگی بر اساس مدل کوله‌پشتی است که با استفاده از پایتون، Flask و SQLite پیاده‌سازی شده است. این سامانه با ارزیابی دقت هر ویژگی و برآورد زمان محاسبه‌ی آن، بهترین زیرمجموعه‌ی ویژگی‌ها را با توجه به محدودیت زمانی مشخص‌شده توسط کاربر انتخاب می‌کند و نتایج را در قالب یک داشبورد تعاملی نمایش می‌دهد.

## قابلیت‌ها
- **انتخاب ویژگی هزینه‌محور** با استفاده از الگوریتم ۰/۱ کوله‌پشتی و مدل رگرسیون لجستیک جهت ارزیابی دقت.
- **پیش‌پردازش داده‌ها** شامل جایگزینی مقادیر گمشده و کدگذاری ویژگی‌های غیرعددی.
- **داشبورد تحت وب** با امکان آپلود دیتاست، مشاهده وضعیت پردازش، نمایش نمودار تحلیل ویژگی‌ها و ثبت تاریخچه اجراها.
- **پایگاه‌داده SQLite** برای نگه‌داری گزارش اجراها و ویژگی‌های منتخب.
- **داونلود خروجی** شامل لیست ویژگی‌های انتخاب شده برای هر اجرا.

## ساختار پروژه
```
optifeat/
├── config.py              # تنظیمات مرکزی پروژه
├── data/
│   └── dataset_loader.py  # بارگذاری و اعتبارسنجی دیتاست
├── modeling/
│   └── evaluator.py       # محاسبه دقت ویژگی‌ها با مدل ML
├── optimization/
│   └── knapsack_selector.py  # حل‌کننده مسئله کوله‌پشتی
├── services/
│   └── pipeline.py        # اجرای کامل فرایند انتخاب ویژگی
├── storage/
│   └── database.py        # مدیریت SQLite
└── web/
    ├── app.py             # Flask app factory
    ├── routes.py          # ثبت بلوپرینت‌ها
    ├── views.py           # کنترلرهای وب
    ├── templates/         # قالب‌های HTML
    └── static/            # فایل‌های CSS و JS
```

## نصب و اجرا
1. ایجاد و فعال‌سازی محیط مجازی (اختیاری اما توصیه‌شده):
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # در ویندوز: .venv\\Scripts\\activate
   ```
2. نصب وابستگی‌ها:
   ```bash
   pip install -r requirements.txt
   ```
3. اجرای برنامه:
   ```bash
   export FLASK_APP=optifeat.web.app:create_app  # در ویندوز از set استفاده کنید
   flask run
   ```
4. مرورگر را باز کرده و به آدرس [http://localhost:5000](http://localhost:5000) بروید.

## نحوه استفاده
1. در داشبورد فایل CSV خود را آپلود کنید.
2. ستون هدف (Label) و محدودیت زمانی را مشخص کنید.
3. پس از اتمام پردازش، ویژگی‌های منتخب، نمودار دقت/هزینه و مراحل پردازش نمایش داده می‌شوند.
4. تاریخچه اجراهای قبلی در بخش تاریخچه قابل مشاهده و دانلود است.

## توسعه
- پیاده‌سازی به صورت ماژولار انجام شده و افزودن مدل‌های ارزیابی یا روش‌های بهینه‌سازی دیگر با توسعه ماژول‌های `modeling` و `optimization` امکان‌پذیر است.
- فایل‌های استاتیک و قالب‌ها با HTML، CSS و JavaScript خالص نوشته شده‌اند و به راحتی قابل سفارشی‌سازی هستند.
